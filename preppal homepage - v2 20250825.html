EPHEMERAL

A. everlasting
B. fleeting
C. powerful
D. essential
E. beautiful
B. fleeting

✅ Correct! The answer is fleeting.

Explanation: Ephemeral means lasting for a very short time.

Synonyms: transitory, momentary, transient

Memory Tip: It sounds like 'a funeral'. Funerals are for things that don't last forever. Sad, but it helps you remember!
